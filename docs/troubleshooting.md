# Troubleshooting

## The menu does not open

Check that:

- BepInEx IL2CPP is installed.
- ElysiumModMenu.dll is inside Among Us/BepInEx/plugins.
- The game was restarted after installation.
- You are pressing Insert or Right Shift.

## The game crashes on launch

Try:

- Updating BepInEx IL2CPP.
- Removing old versions of the mod.
- Checking for incompatible mods.
- Reading the BepInEx console or log file.

## Config changes do not apply

Try:

- Closing the game before editing the config.
- Saving the file with UTF-8 encoding.
- Restarting the game after changes.

## Custom background does not show

Check that the file exists here:

Among Us/BepInEx/config/ElysiumModMenu/MenuBG.png

The image must be a PNG file.

## Still not working

Open an issue on GitHub and include:

- Game version
- Mod version
- BepInEx version
- Steps to reproduce the issue
- Relevant logs, if safe to share
