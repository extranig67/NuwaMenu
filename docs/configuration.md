# Configuration

ElysiumModMenu creates configuration files after the first launch.

## Main configuration file

The main configuration file is usually located here:

Among Us/ElysiumModMenu/ElysiumModMenu.cfg

Some settings may also be generated inside the BepInEx config directory depending on the installed version.

## What can be configured

You can configure:

- Menu keybinds
- Visual options
- Chat settings
- Lobby automation
- Theme and background image
- Diagnostic log reports
- Protection and moderation settings

## Custom background

To use a custom menu background, place a PNG image here:

Among Us/BepInEx/config/ElysiumModMenu/MenuBG.png

Restart the game after changing the image.

## Important

Always close the game before editing configuration files manually.
