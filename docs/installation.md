# Installation Guide

ElysiumModMenu requires a compatible BepInEx IL2CPP installation for Among Us.

## Installation steps

1. Install BepInEx IL2CPP into the Among Us game directory.
2. Launch Among Us once to generate the BepInEx folders.
3. Close the game.
4. Download ElysiumModMenu.dll from the latest GitHub release.
5. Copy ElysiumModMenu.dll into:

Among Us/BepInEx/plugins

6. Start the game.
7. Press Insert or Right Shift to open the menu.

## Finding the game directory

### Steam

Library → Among Us → Manage → Browse local files.

### Epic Games

Library → Among Us → Manage → Open install location.

### Xbox app

Manage → Files → Browse.

## Notes

If the menu does not open, check that BepInEx IL2CPP is installed correctly and that the DLL is placed in the correct folder.
