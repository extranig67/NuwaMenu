# Logs and Diagnostics

ElysiumModMenu can send diagnostic logs when freezes, overloads, or repeated errors are detected.

These reports help identify and fix technical problems.

## What diagnostic logs are used for

Diagnostic reports may help with:

- Freeze detection
- Repeated error detection
- Overload detection
- Stability debugging
- Future release improvements

## Disable diagnostic log reports

Log reporting can be disabled in:

Among Us/ElysiumModMenu/ElysiumModMenu.cfg

Set this option to false:

[ElysiumModMenu.Diagnostics]
EnableAnomalyLogReports = false

## Important

Do not share logs publicly if they contain private session details.
